決定的の変数

  • State
  • Players
  • Actions
  • Transmition Function $SxA \to S$
    • 集合のアレ、わすれた

プレイヤーのSolution（解決策）は、policy$S\to A$によって、定義される
つまり、方針によって、状態$S$で取る行動$A$が定まる。

Value of State

$V(s)=状態sから達成できる最大のスコア$
と定義する。

再帰的には

$V(s) = \max_{s’ \in children(s)} V(s’)$

と示すことができる。
